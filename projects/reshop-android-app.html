ReShop Android App

05/2024 - 08/2024

Java, SQL, HTML, CSS

Project Overview

ReShop is an Android e-commerce application inspired by Depop, designed to provide users with a platform to buy and sell items in a social marketplace environment. As the team lead, I coordinated the development of this prototype from concept to implementation.

Key Features

  • User Authentication: Secure login system with email verification and password recovery
  • Product Listings: Users can create, edit, and delete item listings with photos and descriptions
  • Search Functionality: Advanced filtering by category, price range, and location
  • Messaging System: Real-time chat between buyers and sellers
  • User Profiles: Customizable profiles with ratings and reviews

My Contributions

  • Led a team of 4 developers in an Agile environment with bi-weekly sprints
  • Designed the database schema and implemented SQLite for local storage
  • Integrated Firebase Authentication for secure user sign-up and login
  • Developed the core listing functionality including image uploads
  • Implemented the search algorithm with filtering capabilities
  • Managed Git version control and resolved merge conflicts

Technical Implementation

Frontend

  • Android Studio
  • Java
  • XML for UI layouts
  • Material Design Components

Backend

  • Firebase Authentication
  • Firebase Realtime Database
  • Firebase Storage
  • SQLite for offline caching

Project Outcome

  • Received positive feedback from potential users during beta testing
  • Improved team collaboration skills through effective task delegation
