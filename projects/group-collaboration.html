Group Collaboration System

09/2022 - 12/2022

HTML, JavaScript, CSS

Project Overview

A web-based platform designed to facilitate group work among students, featuring document sharing, real-time chat, and task management functionalities.

Key Features

  • Document Sharing: Upload, download, and collaborative editing of files
  • Real-time Chat: Instant messaging between group members
  • Task Management: Assignment and tracking of group tasks
  • Accessibility: WCAG 2.1 compliant interface

My Contributions

  • Designed and implemented the responsive front-end interface
  • Developed the document sharing functionality with drag-and-drop support
  • Integrated Socket.io for real-time chat features
  • Ensured accessibility compliance (WCAG 2.1 AA standard)
  • Led the team in Agile development with weekly sprints
  • Implemented user authentication and authorization

Technical Implementation

Frontend

  • HTML5
  • CSS3 with Flexbox/Grid
  • JavaScript (ES6+)
  • Responsive Design

Backend

  • Node.js
  • Express.js
  • Socket.io for real-time features
  • MongoDB for data storage

Project Outcome

  • Successfully deployed and used by 50+ students in pilot testing
  • Reduced email communication among group members by 70%
  • Improved task completion rates by 40% through better visibility
  • Recognized by professors as an exemplary student project
